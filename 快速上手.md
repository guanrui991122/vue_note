# 快速上手Typora

## 1.文章结构

在 Typora中，只需按快捷键Ctrl+1，Ctrl+2，……，Ctrl+6，就可以依次输入一级到六级标题。也可以输入#（1-6个） + 空格也可以输入一到六级标题。

```markdown
# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题

**哈哈** //加粗

*呵呵呵* //斜体

***就安心就***
1234567891011121314151617
```
